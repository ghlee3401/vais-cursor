### 7.1 UI 컴포넌트 라이브러리

> 검증된 컴포넌트 라이브러리를 사용하면 일관되고 세련된 UI를 빠르게 구현할 수 있습니다.

| 라이브러리 | 특징 | 적합한 경우 |
|-----------|------|-----------|
| **shadcn/ui** (추천) | Radix UI + Tailwind CSS. 복사-붙여넣기 방식, 커스터마이징 자유도 높음 | Next.js/React + Tailwind 프로젝트 |
| **Ant Design** | 엔터프라이즈급, 대시보드/관리자 화면에 강함 | 관리자 패널, B2B |
| **Material UI (MUI)** | Google Material Design, 풍부한 컴포넌트 | Material Design을 따르는 프로젝트 |
| **Chakra UI** | 간결한 API, 접근성 우선 | 빠른 프로토타입, 소규모 프로젝트 |
| 직접 구현 | 완전한 제어, 의존성 없음 | 디자인 시스템이 이미 있는 경우 |
