### 6.2 실행 가이드

> 아래 명령어를 순서대로 실행하세요. 각 피처는 `plan → ia → wireframe → design → fe+be → check → review` 워크플로우를 따릅니다.

**Step 1: 첫 번째 피처 전체 워크플로우 실행**
```bash
/vais auto {첫번째_피처명}
```
또는 단계별로 진행하려면:
```bash
/vais plan {첫번째_피처명}
/vais ia {첫번째_피처명}
/vais wireframe {첫번째_피처명}
/vais design {첫번째_피처명}
/vais fe+be {첫번째_피처명}
/vais check {첫번째_피처명}
```

**Step 2 이후: 다음 피처로 반복**
```bash
/vais auto {두번째_피처명}
/vais auto {세번째_피처명}
```

> 💡 **팁**: 순차 체이닝으로 한 번에 여러 단계 실행 가능 — `/vais plan:ia:wireframe {피처명}`
