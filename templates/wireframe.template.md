## 5. 컴포넌트 어노테이션

HTML 와이어프레임에 `data-component`, `data-props` 속성을 추가하여
frontend-dev 에이전트가 기계적으로 컴포넌트를 매핑할 수 있도록 합니다.

```html
<div data-component="LoginForm" data-props="onSubmit, isLoading, error">
  <input data-component="EmailInput" data-props="value, onChange, error" />
  <button data-component="SubmitButton" data-props="disabled, loading">로그인</button>
</div>
```
