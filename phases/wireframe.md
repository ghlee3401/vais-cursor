### 🖼 wireframe — 와이어프레임

옵션: `--format ascii|html|mermaid` (기본: ascii), `--device mobile|tablet|desktop|all` (기본: mobile)

1. IA 문서 먼저 읽기
2. **피처 레지스트리 확인** (`.vais/features/$1.json`) — 모든 기능과 화면이 와이어프레임에 반영되었는지 대조
3. 각 화면의 와이어프레임 생성 — 레지스트리의 `screens[]` 기준
4. 반응형 레이아웃 (모바일/태블릿/데스크탑)
5. **컴포넌트 어노테이션**: `data-component`, `data-props` 속성 추가
   ```html
   <div data-component="LoginForm" data-props="onSubmit, isLoading, error">...</div>
   ```
6. `docs/04-wireframe/$1.md`에 설명 문서 저장

#### 출력 형식

**1. ASCII 와이어프레임 (기본)**

터미널에서 바로 확인 가능한 텍스트 기반 와이어프레임입니다.

규칙:
- 박스는 `┌─┐│└─┘` 문자 사용
- 버튼은 `[ Button ]` 형식
- 입력필드는 `[____________]` 형식
- 텍스트 영역은 `{영역 설명}` 형식
- 이미지/아이콘은 `[🖼 설명]` 형식
- 네비게이션은 `[Nav1 | Nav2 | Nav3]` 형식

**2. HTML 와이어프레임**

실제 브라우저에서 확인 가능한 HTML 파일을 `wireframes/$1/` 디렉토리에 생성합니다.
- 각 화면별 `{screen-name}.html` 파일
- `index.html`에 전체 화면 목록과 네비게이션
- Tailwind CSS CDN 사용
- 실제 동작하는 인터랙션 포함 (탭, 모달, 드롭다운 등)

**3. Mermaid 다이어그램**

화면 흐름도를 Mermaid로 생성합니다.

#### 디바이스별 규격

| 디바이스 | 너비 | ASCII 칸 수 |
|----------|------|------------|
| Mobile | 375px | 40칸 |
| Tablet | 768px | 60칸 |
| Desktop | 1280px | 80칸 |

#### ASCII 컴포넌트 라이브러리

```
# 헤더/네비게이션
┌─────────────────────────────────────────┐
│ [🖼] AppName    [Nav1] [Nav2]  [👤 User]│
└─────────────────────────────────────────┘

# 카드
┌──────────────────┐
│ [🖼 이미지 영역] │
│ 제목 텍스트      │
│ 설명 텍스트...   │
│ [ 버튼 ]         │
└──────────────────┘

# 폼
┌─ 폼 제목 ────────────────┐
│ 라벨                      │
│ [_______________________] │
│ 라벨                      │
│ [_______________________] │
│ [ 취소 ]    [ 제출 ]      │
└───────────────────────────┘

# 테이블
┌─────┬──────────┬────────┬───────┐
│ #   │ 이름     │ 상태   │ 액션  │
├─────┼──────────┼────────┼───────┤
│ 1   │ 항목 A   │ ✅     │ [편집]│
└─────┴──────────┴────────┴───────┘

# 모달
┌─────────────────────────────┐
│ 모달 제목              [✕]  │
├─────────────────────────────┤
│ 모달 내용                   │
├─────────────────────────────┤
│         [ 취소 ] [ 확인 ]   │
└─────────────────────────────┘

# 사이드바 + 컨텐츠
┌────────┬────────────────────────────┐
│ 메뉴1  │ {메인 컨텐츠 영역}        │
│ 메뉴2  │                           │
│ 메뉴3  │                           │
└────────┴────────────────────────────┘

# 바텀 네비게이션 (모바일)
┌──────────────────────────────────────┐
│        {화면 컨텐츠}                 │
├──────┬──────┬──────┬──────┬─────────┤
│ 🏠   │ 🔍   │ ➕   │ 💬   │ 👤      │
│ 홈   │ 검색 │ 등록 │ 채팅 │ 마이    │
└──────┴──────┴──────┴──────┴─────────┘
```

#### 어노테이션 규칙

- `(*)` : 필수 입력
- `(→ 화면명)` : 화면 이동
- `(popup)` : 팝업/모달
- `(scroll)` : 스크롤 영역
- `(drag)` : 드래그 가능
- `(auto)` : 자동 갱신
